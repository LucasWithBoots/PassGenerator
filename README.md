# PassGenerator

![Minhas habilidades](https://skillicons.dev/icons?i=ts,react,tailwind)

Um projeto simples em React Native para praticar alguns conceitos importantes, entre eles:
- Tipagem no Typescript
- Modularização em forma de componentes
- Importação de bibliotecas do expo, react e entre outras
- Estilos no TailWind
- Hooks no React, como useEffect e useState
- Comunicação dos componentes por meio de props

Para verificar a segurança da senha utilizei a biblioteca [fast-password-entropy](https://www.npmjs.com/package/fast-password-entropy) para retornar o valor da entropia.
